# Thala 7 Sneaker Store

A simple responsive sneaker store website built using HTML and CSS.

## About The Project

Thala 7 Sneaker Store is a frontend e-commerce website design created to practice HTML and CSS concepts. The project includes a modern landing page, product showcase section, shopping cart UI, and checkout page design.

The website is fully responsive and works across mobile, tablet, and desktop devices.

## Features

* Responsive Navigation Bar
* Hero Section
* Product Showcase
* Product Details Section
* Shopping Cart UI
* Coupon Code Section
* Checkout Form
* Payment Method Section
* Responsive Design
* Hover Effects
* Mobile Navigation Menu
